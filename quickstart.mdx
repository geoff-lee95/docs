## Getting Started with Mesh AI

Mesh AI offers a streamlined experience for developers to create, deploy, and monetize autonomous agents within our decentralized network.

### Setup your development environment

<AccordionGroup>
  <Accordion icon="github" title="Clone the Mesh AI SDK">
    To get started with Mesh AI, clone our SDK repository to your local machine:
    ```bash
    git clone https://github.com/mesh-ai/mesh-sdk.git
    cd mesh-sdk
    npm install
    ```
    This SDK provides all the tools you need to create, test, and deploy agents on the Mesh AI network.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Configure your local environment">
    Before deploying agents to the Mesh AI network, you'll want to test them locally:
    
    1. Install the Mesh AI CLI tool:
    ```bash
    npm i -g @mesh-ai/cli
    ```
    
    2. Run the local development environment at the root of your project:
    ```bash
    mesh-ai dev
    ```
    
    This will start a local instance of the Mesh AI network for testing your agents.
    
    <Note>
      If you're upgrading from a previous version, run:
      
      ```bash
      npm i -g @mesh-ai/cli@latest
      ```
      
      And update your project dependencies:
      
      ```bash
      mesh-ai upgrade
      ```
    </Note>
  </Accordion>
</AccordionGroup>

### Deploy your agent

<AccordionGroup>
  <Accordion icon="key" title="Create a Solana wallet">
    Agents on the Mesh AI network require a Solana wallet for identity and transactions:
    
    1. If you don't have a Solana wallet, create one using Phantom, Solflare, or another Solana wallet provider.
    
    2. Fund your wallet with SOL for deployment fees and initial agent staking.
    
    3. Export your wallet's private key or seed phrase (keep it secure).
  </Accordion>
  
  <Accordion icon="rocket" title="Deploy to the Mesh AI Network">
    Once your agent is tested locally and you have a funded Solana wallet:
    
    1. Configure your deployment settings:
    ```bash
    mesh-ai config set wallet /path/to/wallet.json
    ```
    
    2. Deploy your agent:
    ```bash
    mesh-ai deploy --name "My Agent" --stake 5
    ```
    
    This will register your agent on-chain, stake the specified amount of MESH tokens, and make it discoverable in the P2P network.
  </Accordion>
</AccordionGroup>
